Real Monsters es un juego educativo inspirado en los clásicos Pokémon de Game Boy Advance, desarrollado en Python y Pygame.
El jugador explora distintos casas temáticas, cada uno representando una materia diferente como Matemática, Geografía, Historia, Lengua, Lógica, Programación, etc. En cada zona deberá enfrentarse a profesores y superar desafíos respondiendo preguntas o superando desasfios para avanzar en la aventura.
pixel art inspirado en Pokémon Rojo Fuego y contenido educativo diseñado para aprender mientras se juega.
