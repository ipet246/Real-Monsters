# Manual de Uso:

## Programas Utilizados:

* Python
* Pygame
* Visual Studio Code

## Plataforma Compatible

El juego funciona en computadoras con los programas Visual Studio Code y Python con su extensión de juegos Pygames

## Instalación:

1. Descargar la carpeta del proyecto.
2. Verificar que todos los archivos se encuentren dentro de la misma carpeta.
3. Abrir el proyecto en Visual Studio Code.

## Ejecución:

1. Abrir el archivo principal del juego.
2. Ejecutar el programa.
3. Esperar a que se abra la ventana del juego.

## Controles:

* Flechas o WASD: mover al personaje.
* Enter: confirmar opciones y avanzar diálogos.

## Descripción:

Este juego fue desarrollado en Python utilizando la biblioteca Pygame. El jugador deberá avanzar a través de una aventura educativa enfrentando desafíos y combates basados en preguntas de Matemática. Para progresar será necesario responder correctamente y superar a los distintos enemigos que aparezcan durante la partida.
